# Homework - Shopping Cart

## Setup
* 每个人需要fork repository
* 在自己的repository上进行commit

## Install Environment
Windows: 
``` sh
pip install Flask
```
Mac:
``` sh
pip3 install Flask
```

## Requirements
* 一个**输入框**用于输入item
* 一个**按钮**可以将输入框的内容存入购物车
* 一个**展示**购物车的界面
* 一个**删除**购物车item的按钮

## Hint

``` sh
from flask import Flask, render_template, request, redirect, session, url_for
```
